 Aplicativo Versão 1.0.0 Descrição

Este projeto é um Trabalho de Conclusão de Curso (TCC), que tem como objetivo desenvolver um sistema capaz de informar, via internet, dados meteorológicos obtidos de uma estação Vantage Vue e disponibilizar para o usuário dados e gráficos através de uma aplicação Web e de um aplicativo para o sistema operacional Android.

O projeto é composto por três partes, um aplicativo Web/Android que possui a função de exibir os dados para o usuário, uma API RESTful que é reponsável por armazenar e servir os dados referentes as leituras da estação meteorológica, bem como os dados de usuários, ambos armazenados em MongoDB, e um Driver que faz a comunicação via Socket TCP entre um computador e a estação meteorológica para capturar e então armazenar esses dados no banco de dados através da API.

Este TCC é um subprojeto do projeto Sistema de Gestão da Produção de Insetos Modificados para Regiões Endêmicas, conjunto entre a Moscamed e o Colegiado de Engenharia de Computação da Universidade Federal do Vale do São Francisco - CECOMP.
